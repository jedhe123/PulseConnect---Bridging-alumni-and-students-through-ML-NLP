# PulseConnect – Bridging Alumni & Students Through ML & NLP

## 📖 Introduction

PulseConnect is an AI-powered career guidance and alumni networking platform developed to bridge the gap between students and alumni. The system uses Machine Learning (ML) and Natural Language Processing (NLP) techniques to analyze student profiles, resumes, skills, and interests, providing personalized job recommendations, alumni mentorship suggestions, and career guidance.

The platform enables students to connect with experienced alumni, gain industry insights, receive placement support, and make informed career decisions. It also helps institutions strengthen alumni engagement and improve placement outcomes through intelligent automation.

> **Note:** This project is developed for educational and academic purposes only.

---

## 🎯 Objectives

 - To develop a secure student registration and login system.
 - To automate resume analysis using NLP techniques.
 - To train a machine learning model for job prediction.
 - To recommend personalized company drives for students.
 - To provide a vacancy-sharing platform where alumni can post job openings,
    internship opportunities, and recruitment updates from their organizations.
 - To simplify placement management for administrators.

## ⚙️ Installation & Setup

### Prerequisites

- Python 3.x
- XAMPP
- MySQL
- VS Code
- Git

### Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/PulseConnect.git
cd PulseConnect
```

### Step 2: Install Required Packages

```bash
pip install -r requirements.txt
```

### Step 3: Start XAMPP

Open XAMPP Control Panel and start:

- Apache
- MySQL

### Step 4: Configure Database

- Create a MySQL database.
- Import the database file provided in the project.
- Update database credentials in the project configuration file.

### Step 5: Run the Application

```bash
python app.py
```

### Step 6: Open in Browser

```text
http://localhost:5000
```
